Date | Fecha | Pipe

Pipe para mostrar las fechas

  • Fecha
  • Fecha | date
  • Fecha | date : short
  • Fecha | date : medium
  • Fecha | date : long
  • Fecha | date : MMMM,dd,yyy
  • {{ fecha }}
  • {{ fecha | date }}
  • {{ fecha | date: "short" }}
  • {{ fecha | date: "medium" }}
  • {{ fecha | date: "long" }}
  • {{ fecha | date: "MMMM dd, yyyy" }}
